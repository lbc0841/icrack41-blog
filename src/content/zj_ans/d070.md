## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (2ms, 320KB)**

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int year;
    while(true){
        cin >> year;
        if(year == 0) break;
        
        if((year%4==0 && year%100!=0)||year%400==0) cout << "a leap year\n";
        else cout << "a normal year\n";
    }

    return 0;
}
```
