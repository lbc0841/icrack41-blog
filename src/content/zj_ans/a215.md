## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (2ms, 308KB)**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    while(cin >> n >> m){
        int sum=0, t=0;
        while(true){
            sum += (n + t);
            t++;
            
            if(sum > m) break;
        }
        cout << t << "\n";
    }
    
    return 0;
}
```
