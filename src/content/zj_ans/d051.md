## 解法一

### 🔹 溫度轉換

$F = \frac{9}{5}C + 32$

$C = \frac{5}{9}C \times (F-32)$

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (2ms, 332KB)**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    double f;

    scanf("%lf", &f);
    printf("%.3lf\n", (f-32) * 5/9);
    
    return 0;
}
```
