## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (0.6s, 344KB)**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int a, b, n = 0;
    while(cin >> a >> b >> n){
        int r = (a%b) * 10;
        
        cout << a/b << ".";
        for(int i=0; i<n; i++){
            cout << r/b;
            r %= b;
            r *= 10;
        }
        cout << "\n";
    }

    return 0;
}
```
