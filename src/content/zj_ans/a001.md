## 解法二、scanf

使用字元陣列

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (2ms, 328KB)**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    char s[1000];
    scanf("%s", s);
    printf("hello, %s", s);

    return 0;
}
```
