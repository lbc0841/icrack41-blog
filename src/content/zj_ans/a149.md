## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (2ms, 308KB)**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    
    for(int i=0; i<n; i++){
        string s;
        cin >> s;
        
        int ans = s[0] - '0';
        for(int j=1; j<s.size(); j++){
            ans *= s[j] - '0';
        }
        cout << ans << "\n";
    }
    
    return 0;
}
```
