## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (2ms, 348KB)**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int a, b;
    cin >> a >> b;
    
    if(a & 1) a++;
    if(b & 1) b--;
    
    if(a <= b) cout << (b-a)/2 + 1;
    else cout << 0;
    
    return 0;
}
```
