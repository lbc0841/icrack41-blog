## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (2ms, 316KB)**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, num;
    vector<int> v(3, 0);

    cin >> n;
    for(int i=0; i<n; i++){
        cin >> num;
        v[num % 3]++;
    }

    cout << v[0] << " " << v[1] << " " << v[2];
         
    return 0;
}
```
