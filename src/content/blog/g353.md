## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (3ms, 340KB)**

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string s;
    getline(cin, s);
    stringstream ss(s);
    
    vector<int> nums;
    int num;
    while(ss >> num) nums.push_back(num);
    
    cout << nums[nums.size()/2];
    
    return 0;
}
```
