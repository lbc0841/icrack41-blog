## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (2ms, 328KB)**

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int t, h1, m1, h2, m2, m3;
    cin >> t;
    
    for(int i=0; i<t; i++){
        cin >> h1 >> m1 >> h2 >> m2 >> m3;
        cout << ((h2-h1)*60+m2-m1>=m3 ? "Yes\n" : "No\n");
    }
    
    return 0;
}
```
