## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (3ms, 340KB)**

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int a0, an, d;
    cin >> a0 >> an >> d;
    
    for(int i=a0; i!=an; i+=d){
        cout << i << " ";
    }
    cout << an;
    
    return 0;
}
```
